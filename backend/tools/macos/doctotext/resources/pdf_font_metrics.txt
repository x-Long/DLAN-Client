Binary distribution of doctotext contains informations about 14 core fonts from Adobe.
Those data was obtained here:
http://www.ctan.org/tex-archive/fonts/adobe/afm/

Font metrics are used by PDF parser to compute the boundary of each character
written with a proportional font.
